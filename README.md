# JustMc-Compile-Editor
это компилятор сделаный как дополнение к
компилятору JMCC DONZGOLD он не имеет ничего
нового но работает с всеми версиями
сделан был на скорую руку скачав его вам надо будет
Обязательные пакеты:
pip install pygame
pip install requests

Опциональные (зависят от вашего компилятора):
# Для работы с NBT данными (если используется в компиляторе)
pip install nbt

# Для работы с JSON
# (встроен в Python, не требует установки)


Проверка необходимостей 
# Python
import pygame
import requests
print("Все пакеты установлены успешно!")

Если используете виртуальное окружение:

# BASH
# Создание виртуального окружения
python -m venv venv

# Активация (Windows)
venv\Scripts\activate

# Активация (Linux/Mac)
source venv/bin/activate

# Установка пакетов
pip install pygame requests
